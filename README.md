# 🚦 Traffic Monitoring System (Advanced Backend)

A real-time traffic monitoring system built with Flask, OpenCV, and YOLOv8.

## ✨ Features

- 📡 Real-time video processing
- 🚗 Vehicle detection (YOLOv8 or stub mode)
- 📊 Live metrics:
  - Density
  - Congestion level
  - FPS
  - Vehicle class distribution
- 📈 Rolling history for charts
- 🔴 Live SSE (Server-Sent Events)
- 🎥 MJPEG video streaming
- 🚨 Alert system for high congestion
- 📁 CSV export of session data
- 🧠 Modular architecture (plug real AI models easily)
